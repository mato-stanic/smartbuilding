Korisnik

Korisnik ima mogućnost izbora hoće li koristiti aplikaciju za android mobilne uređaje ili će koristiti web aplikaciju koja mu je dostupa

Ukoliko se koristi android aplikacija dostupne su sljedeće funkcionalnosti: Trenutno upravljanje rasvjetom, Dodavanje novih automatiziranih zadataka, Pregled i brisanje postojećih automatiziranih zadataka

Korištenje web aplikacije omogućuje još neke dodatne funkcionalnosti, kao što su: Slanje poruka administratoru sustava, Dodavanje novih korisnika aplikacije, Mijenjanje korisničkih podataka.

Raspberry Pi

Raspberry Pi je spojen s maketom, te je uz pomoć GPIO pinova na raspberry omogućeno kontroliranje rasvjete makete.

Aplikacija koja je pokrenuta na raspberry pi je povezana s bazom koja se nalazi na serveru. Iz nje dohvaća podatke koje svjetlo treba biti upaljeno, a koje ugašeno.
